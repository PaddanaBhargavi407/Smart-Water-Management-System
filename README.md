# Smart-Water-Management-System
# Overview
An ESP32-based smart water management system that monitors water levels in real time using an ultrasonic sensor. 
The system displays water percentage on an LCD, triggers alerts for low levels, and sends usage data to a cloud server 
for predictive analysis to reduce water wastage.

# Features
- Real-time water level monitoring
- LCD-based water percentage display
- Water usage rate calculation
- Cloud-based prediction via REST API
- Buzzer alerts for low water levels

# Hardware & Software
**Hardware:** ESP32, HC-SR04 Ultrasonic Sensor, I2C LCD, Buzzer  
**Software:**  Thonny software,MicroPython, REST API, I2C Communication,
